# Banking-Information-system-core-java
core java banking information system-internship project
---

##  How to Run the Project
1. Clone or download the repository
2. Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)
3. Compile and run `BankApp.java`
4. Follow the on-screen menu instructions

##  Sample Outputs
- User registration success with account number
- Login success and failure messages
- Deposit and withdrawal confirmation
- Fund transfer confirmation
- Account statement showing transaction history

##  Limitations
- Data is stored temporarily (in-memory only)
- No database or file persistence
- Console-based interface only
